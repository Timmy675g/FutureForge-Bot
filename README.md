# FutureForge Discord Bot

This bot is to help Teenagers or People find their correct Careers and Jobs for the future

## Tools used for this project :
- SQL Database :
  - Current Available Jobs
  - Career

- UI Button from Discord

## Available Discord Commands in the Discord Bot :
- `!start` [ This Command is to start the Discord Bot conversation with the user]
- `!question`[ This will connect user to a bot that will answer user career related questions! ]
